# Deep Learning and Data Science Laboratory

This repository contains Deep Learning and Data Science laboratory experiments implemented using Python, TensorFlow, and Keras.

## Experiments Included

1. Introduction to Keras
2. Backpropagation using TensorFlow
3. Neural Recommendation Systems
4. Convolutional Neural Networks
5. Advanced CNN Models
6. Deep NLP using LSTM
7. Sequence-to-Sequence Models
8. Deep Learning Frameworks
9. Triplet Loss Models
10. Unsupervised Generative Models

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn

## Software Requirements

- Python 3.x
- Jupyter Notebook / Google Colab
- TensorFlow
- Keras

## Objective

The objective of these experiments is to understand and implement various Deep Learning concepts such as:
- Neural Networks
- CNN
- NLP
- Recommendation Systems
- Seq2Seq Models
- Generative Models
